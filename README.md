# SoftComputing

Okruženje: Python 3.7, deuces 0.2, numpy 1.18.1, matplotlib 3.1.3, opencv-python 4.2.0.32, scikit-learn 0.22.1 

Zbog naših promena u deuces paketu radi kompatibilnosti sa Python 3, potrebno virtuelno okruženje se nalazi u okviru projekta, kao venv.7z

Pokretanje skripte se vrši komandom: Main.py --img "putanjadoslike.jpg"

Ako se slika nalazi u okviru main foldera, dovoljno je za putanju staviti samo naziv slike
